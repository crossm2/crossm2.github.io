Divine Desserts
Web Application

The goal of this application is to provide visitors with easy access to divine dessert recipes sourced from around the web and allow them to save their favorites for later reference. I created this web application utilizing the MVC approach with Angular.js framework and the Food2Fork API. To build this application I leveraged many tools including Yeoman, Bower, Grunt and GitHub.
